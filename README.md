# Stack Rush

ブラウザで動く、物理演算ベースの落ちもの積み上げパズルです。

同じ形をくっつけて消すゲームではなく、ランダムに出るブロックをうまく積み上げ、上部のゴールラインへ到達するまでの落下数を競います。

## Play

```bash
npm install
npm run dev
```

## Controls

- `A` / `D` or `←` / `→`: move
- `W` or `↑`: rotate
- `Space` or click/tap: drop
- Mouse/touch: aim
- `P`: pause

## Tech

- Vite
- Matter.js
